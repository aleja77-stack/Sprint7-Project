Análisis de Datos de Venta de Vehículos
|
## Descripción del Proyecto
Esta aplicación web permite analizar datos de anuncios de venta de vehículos. 

## Funcionalidades
La aplicación proporciona las siguiente información:
- **Histograma de precios**: Visualiza la distribución de precios de los vehículos
- **Gráfico de dispersión**: Muestra la relación entre precio y Milas recorridas
- **Filtros interactivos**: Permite mostrar/ocultar gráficos según las necesidades del usuario mediante un checkbox

## Programas y librerias utilizadas
- Python
- Streamlit
- Pandas
- Plotly

## Cómo ejecutar la aplicación
1. Clona este repositorio
2. Instala las dependencias: `pip install streamlit pandas plotly`
3. Ejecuta la aplicación: `streamlit run app.py`

## Datos
El proyecto utiliza un conjunto de datos de anuncios de venta de vehículos (`vehicles_us.csv`)
